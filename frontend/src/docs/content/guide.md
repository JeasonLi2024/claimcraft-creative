# 从上传证据到导出材料

按下面的步骤走一遍，即可完成一次完整的维权材料整理。

## 快速开始

1. **注册并登录** — 首次使用先注册账户；也可从示例案件快速体验完整流程。
2. **创建案件** — 在「我的案件」新建案件，填写标题、纠纷类型，选择维权投诉或商家反证模式。
3. **上传证据** — 在「证据管理」上传聊天、订单、物流、付款等图片；纯物证可标记并补充说明。
4. **发起工作流分析** — 进入「工作流分析」，选择参与分析的证据并开始，实时查看各阶段产物。
5. **人工复核与介入** — 当系统提示需要确认时，在介入面板核对字段或补充信息后提交继续。
6. **核对时间线** — 在「时间线」检查事件顺序与关联证据，可直接编辑修正。
7. **生成并编辑文书** — 在「投诉文本」或「反证答辩」页逐段编辑、AI 重写并核对法律依据。
8. **隐私打码复核** — 在「脱敏打码」逐项复核文本风险与图片遮罩，确认后再对外分享。
9. **导出材料** — 在「导出」选择输出格式；注意原图与隐私版的区别，按接收对象谨慎选择。

## 关键提示

:::info{title="结果由你确认"}
系统会主动标出低置信度或需要核对的信息。生成文书前，请确认事实、金额与时间无误。
:::

:::warn{title="分享前先脱敏"}
PDF / Word / ZIP 默认使用原图，可能包含未脱敏的隐私信息；对外分享前请在脱敏打码页逐项复核。
:::

:::success{title="随时可恢复"}
分析中断或刷新页面后可继续；失败的图片打码可单张重试，无需从头再来。
:::

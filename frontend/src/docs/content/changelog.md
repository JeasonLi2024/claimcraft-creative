# 版本更新

按主题归纳的主要迭代（依据项目提交记录整理）。

## 隐私与安全导出

- 完善隐私打码：统一扫描证据描述、OCR 文本、抽取字段、时间线与最新文书，按来源分组并标注风险等级
- 单张图片可重试打码，卡住的处理状态可恢复；打码图改为无损输出并移除元数据
- 重构导出中心：新增导出格式选择与原图/隐私影响说明、下载前隐私确认

## 文书工作台

- 新增文书工作台组件，投诉页与反证页支持逐段编辑、AI 重写与版本管理
- 工作流分析页聚焦全过程展示，文书编辑统一收敛到对应文书页
- 统一文书产物字段，追加法律依据与署名

## 工作流与质量门控

- 引入工作流输入质量校验三道门控，低质量输入及时提示或拦截
- 支持工作流阶段安全暂停、用户编辑参与与终止；修复连接状态与终止/暂停行为
- 重构时间线系统并修复时区一致性问题

## 检索与实时

- 法律 RAG 新增 BM25 检索、重排配置与向量缓存
- 引入 WorkflowRun SSE 事件流与文书版本管理，支持断线续传与刷新恢复
- OCR 多策略级联优化，非瞬时错误（如文件缺失）快速失败，避免节点超时

## 基础与部署

- 新增案件材料 Word 文档导出
- 首页导航栏新增 GitHub 项目链接，优化首页视觉与样式
- 实现共享数据库架构，合并正式/测试实例；完善部署配置

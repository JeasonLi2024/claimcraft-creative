# ClaimCraft 维权材料工坊

把零散的聊天记录、订单、物流与付款凭证，整理成结构化、可核对、可导出的维权材料。

## ClaimCraft 是什么

ClaimCraft 是一个面向普通消费者的**维权材料整理与文书生成工具**。你上传证据后，系统会依次完成材料理解、事实核对、案件组织与文书生成，最终产出结构化的时间线、投诉/答辩文书以及可分享的导出文件。所有原始材料始终保留，关键结论均由你确认。

:::info{title="定位说明"}
ClaimCraft 提供的是材料整理与文书生成的辅助能力，不构成法律意见；自动识别可能存在遗漏或误报，请在对外提交前人工核对。
:::

## 核心能力

### 账户与案件

- 注册登录，按案件维度管理材料，支持维权投诉与商家反证两种模式
- 内置示例案件，便于快速了解完整流程

### 证据与产物

- 上传聊天截图、订单、物流、付款凭证等图片证据，自动 OCR 与字段抽取
- 纯物证图片可跳过 OCR，仅作留存与说明
- OCR 与字段抽取结果支持人工校正，原始材料始终可回看

### 实时工作流

- 一键发起分析，实时查看各阶段进度与节点产物
- 需要确认的信息会明确提示复核，可随时暂停、介入或终止
- 断线自动重连、刷新可恢复进度

### 隐私打码

- 扫描文本与图片中的手机号、身份证号、结构化地址等敏感信息并给出遮罩建议
- 逐张复核图片打码，失败可重试，自动识别不替代人工检查

### 导出

- 支持文本预览、ZIP 原始材料包、PDF 与 Word 多种输出
- 明确区分「原始证据版」与「隐私保护」的图片策略，避免误分享

## 智能分析工作流

案件分析由四个业务阶段串联完成，每个阶段都会产出可查看的中间产物：

1. **材料理解** — 证据预分类、OCR 识别与证据分类，读懂每份材料是什么。
2. **事实核对** — 抽取关键字段（金额、时间、单号等）并在必要时请你人工审核确认。
3. **案件组织** — 构建事实时间线与证据链，关联法律依据。
4. **文书生成** — 生成结构化的投诉书或反证答辩书，支持逐段编辑与重写。

## 技术亮点

- 基于 LangGraph 编排的多节点工作流，支持人在环（HITL）审核与阶段暂停
- 多策略 OCR 级联（云端 PaddleOCR-VL / 多模态大模型 / 本地兜底），非瞬时错误快速失败
- 法律 RAG 三阶段检索（BM25 + 向量 + 融合），为文书提供法条与平台规则依据
- SSE 实时事件流 + 权威快照，前端进度即时更新、断线可续传
